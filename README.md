# Patient Medical History Management Website

A static, responsive personal portfolio and medical history management application built with HTML5, CSS3, and Vanilla JavaScript.

## Features
- **Portfolio Showcase**: Introduction and Contact pages.
- **User Authentication**: Simulation of Login and Signup (session based).
- **Dashboard**: View and Add medical history records.
- **PWA Ready**: Installable with offline fallback.
- **Responsive Design**: Optimized for Mobile, Tablet, and Desktop.

## Usage
1. Open `index.html` in your browser.
2. Navigate to "Login" to create a new ID.
3. Once logged in, use the Dashboard to manage medical records.

## Technologies
- HTML5
- CSS3 (Flexbox/Grid, Variables)
- JavaScript (ES6+, LocalStorage)
